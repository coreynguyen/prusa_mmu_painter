## Release Notes Template

### v1.0.0

**Features:**
- Load OBJ and 3MF models with UV coordinates
- Texture projection with adjustable subdivision (0-10 levels)
- Color quantization (Uniform, K-Means, Median Cut, Octree)
- Interactive 3D painting with undo/redo
- Export PrusaSlicer/OrcaSlicer compatible 3MF files
- Full MMU segmentation format documentation

**Requirements:**
- Windows 10/11
- .NET 8.0 Runtime
